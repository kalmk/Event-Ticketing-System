## Event Ticketing System

- A backend API for managing events, venues, organizers, different types of tickets, and the attendees who book them.

## Screenshots

1. POST /api/organizers (Create a new organizer)


2. POST /api/venues (Create a new venue)


3. POST /api/events (Create a new event)


4. GET /api/events (List all upcoming events)


5. GET /api/events/{id} (Get event details with ticket types)


6. POST /api/attendees (Register a new attendee)


7. POST /api/bookings (Book a ticket)


8. PUT /api/bookings/{id}/cancel (Cancel a booking)


9. GET /api/events/{id}/revenue (Get a total revenue for an event)


10. GET /api/attendees/{id}/bookings (Get all bookings for an attendee)
